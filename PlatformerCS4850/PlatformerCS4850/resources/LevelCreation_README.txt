The levels are made as follows:

WIDTH (in tiles)
HEIGHT (in tiles)
TILES_START
<Array of numbers with index mapping to a sprite in a sprite sheet>
TILES_END
TILEPROPS_START
<Array of numbers with the following mapping:
1 = Solid (collidable wall/ground)
2 = Win (Touch this tile to win)>
TILEPROPS_END
SPAWNS_START
<Array of numbers with the following mapping:
1 = Player spawn point
2 = Enemy spawn point
3 = Pickup A spawnpoint>
SPAWNS_END
